What do we do here

New Contract

This feature enables you to create new contract records. Add and edit contract details and navigate to create associated purchase order records. Click on the links below to open the collapsible content:

Steps

As seen in the picture above, the contract details will be filled out with their respective fields. The user can also add additional details regarding the contract such as: request for proposal(rfp), account classification(CoPA, DE, USDC), and a simple description. There is a save and a save PO option at the top. If you select the save button, it will allow you to continue to the page that includes the update feature and revision details. If you select the save PO option, you will be prompted to fill out details about a purchase order for that specific client. Then, this page will lead into a reseource setup page and make the user create an initiative. The adding an initiatives page will lead to an add artifact page. User will be prompted to either create another artifact or they can save what they have currently.

As seen in the picture above, users can edit contract details after saving that contract in case of errors in the previous save. Once user clicks update, the screen will pop up asking the reasoning for the update to be logged in the revision history.
